## Installation

* Native app
   * Cocoapods

        Add the following to your project's Podfile:
        ```ruby
        pod 'ObjCNavigationController'
        ```

   * Carthage

        Add the following to your project's Cartfile:
        ```ruby
        github "ouyanghuacom/ObjCNavigationController"
        ```
